# 🚀 QR Code Generator

A modern and responsive QR Code Generator built with React. Generate, customize, preview, and download QR codes instantly with support for custom colors, adjustable size, dark mode, and history tracking.

## ✨ Features

* 🔳 Generate QR Codes instantly
* 🎨 Custom QR Code Colors
* 🖼️ Custom Background Colors
* 📏 Adjustable QR Code Size
* 🌙 Dark / Light Mode Toggle
* 📥 Download QR Code as PNG
* 🕒 Recent History Tracking
* 📱 Fully Responsive Design
* ⚡ Fast and Lightweight
* 🎯 Modern UI/UX

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/prabhatrana666/qr-code-generator.git
```

Navigate to the project:

```bash
cd qr-code-generator
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm start
```

---

## 🚀 Build for Production

```bash
npm run build
```
